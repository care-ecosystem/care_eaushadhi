# Installation

## Stable release

To install care_eaushadhi, run this command in your terminal:

```sh
pip install care_eaushadhi
```

## From source

The source files for care_eaushadhi can be downloaded from the [Github repo](https://github.com/care-ecosystem/care_eaushadhi).

You can either clone the public repository:

```sh
git clone git://github.com/care-ecosystem/care_eaushadhi
```

Or download the [tarball](https://github.com/care-ecosystem/care_eaushadhi/tarball/master):

```sh
curl -OJL https://github.com/care-ecosystem/care_eaushadhi/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd care_eaushadhi
pip install .
```
